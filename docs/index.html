Lightbox

Blender study add-on for various utility functions.

Github | Report an Issue

Installation

  1. Open Blender.
  2. In the top bar, open the Edit menu.
  3. Select Preferences window.
  4. On the left side of the Preferences window, choose the Add-ons tab.
  5. In the top right of the Preferences window, click down arrow, then click the Install button.
  6. Browse to the Lightbox .zip file you downloaded.
  7. In the bottom right of the file browser, click Install Add-on.
  8. You should now see Lightbox in the list of add-ons.
  9. Tick the box to the left of the add-on name to enable it.
  10. On your 3D Viewport press N to open context menus.
  11. You should be able to see Lightbox UI on right of your editor.

Features

  • EEVEE Clay Material Override
    Overrides all materials within the scene with a basic clay material. Toggles back to the original materials on button click.
    Access   View3D > Lightbox > Toggle Clay
  • Focus Selected on Outliner
    Focuses selected object on View3D on outliner on complex scenes.
    Access   View3D > Lightbox > Focus Outliner
